# Port Scanner

A simple Python-based port scanner that scans a target host and identifies open ports using the socket module.

## Features

- Scan open ports on a target host
- Fast and lightweight
- Built with Python
- Beginner-friendly cybersecurity project

## Technologies Used

- Python
- Socket Module

## Usage

1. Clone the repository:

```bash
git clone https://github.com/ashu-sec/Port_Scanner.git
```

2. Navigate to the project directory:

```bash
cd Port_Scanner
```

3. Run the script:

```bash
python Port_Scan.py
```

## Example Output

```
Scanning localhost...
Port 22 is open
Port 80 is open
Port 443 is open
```
